Technical Information

The Software has fuctionalities to store and keep track
of employee data. The fuctionalities are:
|| Store: The user can enter a Information about new employee such as NAME, POSITION, SSN,
ADDRESS, EMAIL, PHONE, SKILLS

|| Import: The user can Load a pre-existing database

|| Search: The user can search for an employee in a database

|| Delete: The user can delete an employee of the database

|| Show: The user can show the loaded database.
